1. **ps** 查看进程信息
	- -f 显示进程详细信息
	- -e 显示所有进程信息
	- --forest  显示进程关系
2.  command & 如果command一个前台程序,加上&程序就会放到后台执行
3. **nohup** 如果希望一个进程在关闭终端后依然在后台运行,那么在启用时要加上nohup
	- 如果忘记了可以暂停程序(ctrl+z)在终端上运行`disown` 命令然后恢复之前暂停程序
4. **kill -9** 结束进程

	- kill -9 1420
	- 结束id为1420的进程

5.  **pstree** 查看进程树
	- yum -y install psmisc
