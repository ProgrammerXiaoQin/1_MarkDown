#### 1. 过滤表达式规则
1. IP地址过滤
	- 匹配源IP地址 `ip.src == 192.168.0.102`
	- 匹配目的IP地址 `ip.dst == 192.168.1.202`
	- 匹配源或目标IP `ip.addr == 192.168.0.17`

2. MAC地址过滤
	- 匹配源MAC地址 `eth.src == 3C-F0-11-EB-55-B4`
	- 匹配目的MAC地址 `eth.dst == 3C-F0-11-EB-55-B4`
	- 匹配源或目标MAC `eth.addr == 3C-F0-11-EB-55-B4`

3. 端口号过滤
	- 匹配TCP端口 `tcp.port == 80`
	- 匹配TCP源端口 `tcp.srcport ==80`
	- 匹配TCP目标端口 `tcp.dstport ==80 `

4. 协议过滤 
	1. `icmp,arp,dhcp,http,https,tcp,udp` 等

5. 逻辑运算符 `||` 或, `!=` 不等于,`and` 与, `!` 非

 

#### 2. 例
1. 过滤指定域名的DSN包： `dns.qry.name == www.huawei.com` 
2. 根据IP地址过滤数据包：`ip.addr == 183.223.14.106 or ip.addr ==183.223.14.106` , `ip.dst == 183.223.14.106 or ip.dst == 183.223.14.108`
